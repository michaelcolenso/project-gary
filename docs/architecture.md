# OpportunityForge v1 Architecture

## Mission

OpportunityForge answers: **What public-data-backed business should I build next?**

It optimizes for distribution, trust, proprietary data leverage, domain expertise, commercial access, and customer acquisition rather than engineering effort.

## Pipeline

```text
Data Sources
  -> Scout Agent
  -> Dataset Catalog
  -> Profiler Agent
  -> Decision Agent
  -> Buyer Agent
  -> Monetization Agent
  -> Opportunity Engine
  -> Validation Agent
  -> Opportunity Graph
  -> Dossier Generator
  -> User Dashboard
```

## Scoring model

Opportunity score is the product of five auditable sub-scores:

```text
Demand × Decision Value × Monetization Ease × Defensibility × Strategic Fit
```

Engineering effort is deliberately excluded. Friction is tracked separately for risk visibility, not ranking.

## Friction profile

- Data friction
- Trust friction
- Distribution friction
- Sales friction
- Regulatory friction
- Maintenance friction

## Dossier sections

Every opportunity dossier includes executive summary, target user, decision, data sources, demand analysis, buyer analysis, monetization analysis, competitive landscape, score breakdown, friction profile, MVP recommendation, 30-day validation plan, risks, win/fail cases, and next actions.
